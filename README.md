# Golang Rest API Project Starter

## SYSTEM REQUIREMENTS
- [docker & docker-compose](https://www.docker.com/get-started)
- [golang](https://golang.org/doc/install) version 1.11.4 or newer

## DEVELOPMENT
- `make dev`

## DEPENDENCIES
- `make deps`

## TESTS

### UNIT TESTS
- `make test` or `go test -v ./..`
